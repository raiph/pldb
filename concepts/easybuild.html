Easybuild is an open source application created in 2014 by Kenneth Hoste.

Source code:

EasyBuild is a software build and installation framework that allows you to manage (scientific) software on High Performance Computing (HPC) systems in an efficient way.

  • Easybuild is developed on GitHub and has 377 stars
  • Easybuild first developed in https://github.com/easybuilders/

  • GitHub supports syntax highlighting for Easybuild
